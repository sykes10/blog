<objective>
Deliver a table of contents with working anchor links for long Pattern posts (SITE-04). This satisfies the phase-goal capability "a visitor reading a long post can jump to any heading via a table of contents with anchor links."

Purpose: Long-form Pattern posts (the 8-section template) are hard to navigate without jump links. The TOC gives readers section navigation and a scroll-spy active-heading indicator, reusing the `rehype-slug` heading ids already generated by the Phase 1 MDX pipeline — the one new piece of plumbing is extracting the heading list from the raw MDX so the TOC links match those ids exactly.

Output: `extractHeadings()` helper in `lib/toc.ts`; a `"use client"` `<TableOfContents>` component driven by `IntersectionObserver`; integration into `app/patterns/[slug]/page.tsx`; and a Playwright spec asserting TOC link presence and click-to-anchor.
</objective>
